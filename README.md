# Dashboard Interattiva Vendite

## Descrizione

Questo progetto contiene una dashboard interattiva per l'analisi delle vendite di un'azienda retail/e-commerce, sviluppata in Power BI. La dashboard permette di monitorare KPI principali, analizzare trend temporali e confrontare performance per prodotto, regione e canale.

## Obiettivi

* Monitorare il fatturato, margine, numero ordini e ticket medio.
* Identificare le categorie e i prodotti più performanti.
* Analizzare la performance geografica e dei canali.
* Supportare decisioni strategiche con visualizzazioni interattive.

## Come usare

1. Aprire il file `pbix/raeh.pbix` in Power BI Desktop.
2. I dati sono presenti in `data/Online_Retail.xlsx`.
3. Consultare la documentazione completa in `docs/Dashboard_Interattiva_Vendite.pdf`.
